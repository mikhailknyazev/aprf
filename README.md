# APRF article companion repository

## What is this repository for?

This repository publishes the companion material that adds value after the
reader has already read the article:

- the applications/workloads source
- the simplified workload and GitOps manifests
- a small set of simplified, explanatory deploy roles

It does not publish the article manuscript itself, the broader APRF runtime, or
the raw internal APRF execution tree.

## Start here

Use this order:

1. Read the published article
2. Read all files in this repository as you wish

Thank you!
